# 🚀 Projeto React + Vite

Este é um projeto frontend moderno desenvolvido com **React 19** e **Vite**, estruturado para oferecer performance, estilização eficiente.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza as seguintes bibliotecas e ferramentas:

* **Core:** [React 19](https://react.dev/) & [Vite](https://vitejs.dev/)
* **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/) (estilização via classes utilitárias)
* **Roteamento:** [React Router 7](https://reactrouter.com/) (navegação entre páginas)
* **Ícones:** [Lucide React](https://lucide.dev/) (pacote de ícones leves)
* **API:** [Axios](https://axios-http.com/) (requisições HTTP para o backend)
* **Autenticação:** [@react-oauth/google](https://www.npmjs.com/package/@react-oauth/google) (integração com Login do Google)

---

## 📋 Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:
* [Node.js](https://nodejs.org/) (Versão 18 ou superior)
* Um gerenciador de pacotes (NPM ou Yarn)

---

## ⚙️ Como Executar o Projeto

Siga os passos abaixo para rodar a aplicação localmente:

### 1. Instalar as dependências
Abra o terminal na pasta raiz do projeto e execute:
```bash
npm install
npm run dev
```

### 2. Acessar a aplicação

O Vite informará uma URL no terminal (geralmente http://localhost:5173). Basta clicar no link ou colá-lo no seu navegador.
